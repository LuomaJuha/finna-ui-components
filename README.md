# Finna UI Component Library
## Installation
Copy the repository
```
git clone git@github.com:mikkojamG/ui-component-library-proto.git
```
Install development dependencies and Pattern Lab.
```
yarn
```
or
```
npm install
```

Start the development server with hot reloading
```
yarn serve
```
or
```
npm run serve
```
