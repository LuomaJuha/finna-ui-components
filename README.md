# Finna UI Component Library
## Installation
Copy the repository
```
git clone
```
Install development dependencies and Pattern Lab.
```
yarn
```
or
```
npm install
```

Start the development server with hot reloading
```
yarn serve
```
or
```
npm run serve
```
